# Supply Chain Dataset – Data Dictionary

| Column Name     | Description |
|-----------------|-------------|
| Order_ID        | Unique identifier for each order |
| Order_Date      | Date when the order was placed |
| Ship_Date       | Date when the order was shipped |
| Delivery_Date   | Date when the order was delivered |
| Customer_ID     | Unique ID of the customer |
| Segment         | Customer segment (Consumer, Corporate, Home Office) |
| Region          | Geographic region where order was placed |
| Category        | Product category |
| Sub_Category    | Product sub-category |
| Sales           | Total revenue generated from the order |
| Profit          | Profit earned from the order |
| Quantity        | Number of products ordered |
| Shipping_Mode   | Shipping method (Standard, First Class, etc.) |
| Shipping_Delay  | Number of days between ship date and delivery date |
